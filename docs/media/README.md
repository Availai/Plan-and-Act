# README Media Checklist

Bu klasör, proje README'sinde kullanılacak ekran görüntüleri ve demo dosyaları için ayrılmıştır.

## Gerekli Dosyalar

- `today-dashboard.png`
- `add-plan-sheet.png`
- `reflection-screen.png`
- `momentum-screen.png`
- `stuck-screen.png`
- `planandact-demo.gif`

## Önerilen Çekim Akışı

1. Uygulamayı temiz örnek veriyle açın.
2. `Today Dashboard` ekranını alın.
3. `Yeni Plan` bottom sheet'ini açıp ikinci ekran görüntüsünü alın.
4. Bir planı tamamlayıp `Reflection` ekranını alın.
5. Streak görünürken `Momentum` ekranını alın.
6. Bir görevi birden fazla erteleyip `Stuck Detection` ekranını alın.
7. Tüm akışı 10-20 saniyelik tek bir GIF olarak kaydedin.

## Çekim Notları

- Aynı tema ve aynı cihaz boyutuyla çekim yapın.
- Gerçekçi ama kısa örnek plan isimleri kullanın.
- GIF'de şu akışı koruyun: uygulamayı aç -> plan ekle -> dashboarda dön -> tamamla veya ertele -> ilgili ekran değişimini göster.
