# Plan and Act
This application allows users to create daily plans and sends a special notification when the time for the plan arrives.

## Key Features
1. Planning and Scheduling.
2. Smart Notifications.
3. Wisdom Engine: Generates contextual advice from the lives and books of people (Atatürk, Marcus Aurelius, etc.) according to the content of the plan.

## Technology Stack (Planned)
- Frontend: Flutter (Recommended because it is mobile-focused)
- Backend/Database: Firebase (For fast and easy ad management)
